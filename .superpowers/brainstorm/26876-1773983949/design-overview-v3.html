How it works under the hood

LAYER 1 — Mirror div (behind, pointer-events: none)
Renders colored token spans. Each token (e.g. "from: sarah@acme.co") is a single span with one background. Same font, size, padding as the input above it.
LAYER 2 — Real <input> (transparent text, on top)
Handles cursor, selection, mobile keyboard. Text color is transparent so mirror colors show through.
LAYER 3 — Ghost span (faded, after typed text)
Shows the remaining contact address faded. Appears inside the same token background as the typed portion. Tab accepts, ↑↓ cycles.
