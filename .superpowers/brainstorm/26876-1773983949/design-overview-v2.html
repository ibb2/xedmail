How it works under the hood

LAYER 1 — Mirror div (behind, pointer-events: none)
Renders colored <span> tokens. Same font, size, padding as input. User never interacts with it directly.
LAYER 2 — Real <input> (on top, transparent text color)
Receives all keystrokes, handles cursor, selection, mobile keyboard. Text is transparent so the mirror shows through.
LAYER 3 — Ghost span (absolute, faded)
Shows the contact completion after the cursor. Calculated from cursor position + contact match. Tab accepts it.
