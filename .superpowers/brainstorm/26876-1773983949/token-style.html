How should search tokens look?

When you type "from:" or "unread", the token gets highlighted inline. Which style feels right for June's dark copper theme?

🔍 from: sarah@acme.co unread project update |

A. Discrete Chips

Each recognized token becomes a rounded pill/chip. Clear boundaries, easy to delete individually. More structured feel.

🔍 from: sarah@acme.co unread project update |

B. Inline Color Highlighting

Text stays as plain text but recognized keywords change color. Feels like Fantastical's natural language input. Lighter, faster to type.

🔍 from: sarah@acme.co unread project update |

C. Hybrid (Recommended)

Colored text with a very subtle background tint — best of both worlds. Color provides meaning, soft background groups the token visually without breaking typing flow. Closest to Fantastical.

Proposed Color Mapping

● from: (contact) ● read / unread (status) ● today / yesterday / date (time) ● plain keywords (default)
